# MERN Chat App

A real-time chat application built using the MERN Stack.

## Features

- User Authentication (JWT)
- Real-time Messaging with Socket.IO
- Online/Offline Status
- Responsive UI
- MongoDB Database
- Secure Password Hashing
- Modern React + Tailwind CSS UI

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Socket.IO
- Tailwind CSS
- Zustand

## Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
